# Smart-Aquaculture-An-IoT-Based-Monitoring-and-Management-System

# Features
Real-Time Monitoring: Continuous monitoring of water parameters like temperature, pH, dissolved oxygen, and turbidity.
IoT Connectivity: Data is transmitted to a cloud dashboard for real-time visualization and analytics.
Automated Alerts: Sends notifications (SMS/email/app) when water parameters go out of the safe range.
User Interface: LCD or web-based interface to check live water conditions and system status.

# Hardware Used
Microcontroller 
Temperature, pH, dissolved oxygen, and turbidity sensors
LCD display for local monitoring
IoT module (Wi-Fi) for cloud connectivity
